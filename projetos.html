StylistGo

O StylistGo é um aplicativo web criado no Base 44 com o objetivo de avaliar o look de uma pessoa com base na ocasião. O usuário envia uma foto e a IA estilista profissional fornece uma avaliação automática (nota + feedback + dicas de estilo).

Tecnologias: Base 44 (no-code), OpenAI API.
Desafio: Criar um app interativo capaz de avaliar o look de um usuário conforme a ocasião (ex: reunião, festa, encontro), integrando inteligência artificial e lógica condicional sem uso de backend tradicional.
Resultado: Desenvolvido um MVP totalmente funcional dentro do Base 44, capaz de processar descrições de looks e gerar feedbacks personalizados de estilo em tempo real. O projeto validou o uso de IA generativa dentro de uma plataforma no-code, mostrando o potencial de criar soluções inteligentes e visualmente modernas sem programação manual.

Automação de Agentes de IA

Desenvolvi um sistema de chat de inteligência artificial integrado ao WhatsApp para clientes, capaz de responder automaticamente aos clientes e gerar relatórios semanais sobre leads e interações.

Tecnologias: N8n, OpenAI API, Whatsapp API, Google Sheets .
Desafio: Criar agentes de inteligência artificial capazes de executar tarefas automatizadas, processar dados e gerar respostas contextualizadas, integrando fluxos complexos de forma escalável.
Resultado: Desenvolvido um sistema funcional de agentes de IA que automatiza processos repetitivos, gera respostas inteligentes e permite integração entre diferentes serviços. O projeto demonstrou a viabilidade de criar automações sofisticadas com IA, aumentando eficiência e abrindo caminho para soluções personalizadas em múltiplos fluxos de trabalho.

Portfólio DEV Front-End e Automação de IA

Desenvolvi um portfólio pessoal para apresentar meus projetos e habilidades nas áreas de desenvolvimento front-end e automação com inteligência artificial. O objetivo foi criar um espaço moderno, interativo e responsivo que transmitisse minha identidade profissional e destacasse minha experiência prática.

Tecnologias: HTML5, CSS3, JavaScript, GitHub.
Desafio: O principal desafio foi construir um layout responsivo e fluido, com estrutura clara de navegação, organização dos projetos e integração com repositórios do GitHub, garantindo uma experiência agradável tanto em desktop quanto em dispositivos móveis.
Resultado: Criei um portfólio visualmente moderno, otimizado e responsivo, capaz de apresentar meus projetos e competências de forma profissional. O projeto reforçou minha identidade como desenvolvedor front-end e especialista em automações com IA, servindo como vitrine estratégica para oportunidades de trabalho e parcerias.
